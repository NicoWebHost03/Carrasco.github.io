ACERCA DE MIS ESTUDIOS O TRABAJO

Por ahora no estoy trabajando, ahora me estoy dedicando a mis estudios, en la carrera de Desarrolo de sistemas de informacion que basicamente se refiere a la computacion informatica, aplicando conocimientos en leguajes de programacion y otras servicios mas en los cursos.

La carrera de Desarrollo de Sistemas de Información también conocida como Desarrollo de Software o Ingeniería de Software se trata de enfocar en la formación de profesionales capaces de diseñar, desarrollar, implementar y mantener sistemas de información y software. A traves de esta carrera combina conocimientos de ciencias de la computación, ingeniería de software y gestión de proyectos para crear soluciones tecnológicas que aborden las necesidades y problemas específicos de las organizaciones y usuarios.

Java: Es utilizado en el desarrollo de aplicaciones móviles Android y sistemas embebidos. Python: Se utiliza en el desarrollo web, aplicaciones científicas, análisis de datos y scripting. C#: Es el lenguaje de programación principal utilizado para el desarrollo de aplicaciones en el entorno de desarrollo Microsoft .NET. JavaScript: Es esencial para el desarrollo web, ya que se utiliza para crear interacciones dinámicas y en tiempo real en el navegador. PHP: Es un lenguaje utilizado para el desarrollo web y es especialmente popular para la construcción de aplicaciones basadas en WordPress. SQL: Aunque no es un lenguaje de programación en sí mismo, es esencial en el desarrollo de sistemas de información para gestionar bases de datos y consultas. Ademas, también en esta carrera estan relacionados con el desarrollo web, como HTML, CSS, Angular, React, Node.js, entre otros.

Sirve para desarrollar soluciones tecnológicas que brinden servicios, funciones o contenido a los usuarios a través de dispositivos móviles (como teléfonos inteligentes y tabletas) o a través de navegadores web en computadoras.
